2017 Programming

The code behind the robot.

This was our first year programming the robot in Java. Drew setup Java by himself in about a week, whereas some other teams spent up to a summer trying to set it up. Previously, the team has used Labview, a drag-and-drop application for programming. Drew wrote all of the basic code for the robot, with some help from Matthew, our new programming mentor.

With this code, the robot was driving very easily and all of the pneumatics were working smoothly. We had enough code to drive the robot pretty much as soon as the robot was built. Michael then wanted to join in with the programming and asked what the drive team would want for each of the buttons, such as toggles and single presses. Once the drive team had the controls as they preferred, the programming team went on to try to get cameras and autonomous modes working.
